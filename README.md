# 🇧🇷 BR Document Utils

Biblioteca em Python para **validação e geração de documentos brasileiros**, como CPF, CNPJ e outros identificadores comuns no Brasil.

> ⚠️ Projeto em desenvolvimento.

---

## 🚀 Funcionalidades

- Validação de CPF
- Validação de CNPJ
- Geração de CPF válido (para testes)
- Geração de CNPJ válido (para testes)
- Estrutura preparada para adicionar novos documentos brasileiros

---

## 🧠 Objetivo do projeto

Este projeto foi criado com fins de estudo e prática de:

- Lógica de validação de documentos reais
- Algoritmos de dígitos verificadores
- Estruturação de bibliotecas em Python
- Boas práticas de código reutilizável
- Publicação de pacotes no futuro (PyPI)

---

## 🛠️ Tecnologias utilizadas

- Python 3.10+
- Módulos padrão da linguagem
